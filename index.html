built

AI-Powered Job Search Workflow

Summary

  • Built an end-to-end n8n workflow that scrapes job listings daily from LinkedIn and Indeed using Apify actors, with configurable inputs so others can adapt it easily
  • Implemented AI-based job matching using Gemini 2.5 Flash, assigning each role a relevance score out of 10 based on predefined criteria to prioritize higher-fit opportunities
  • Optimized API usage to handle Gemini's rate limits by batching jobs into a single JSON payload per cycle — reducing API calls by ~80%
  • Added recruiter detail extraction to surface contact information for direct outreach
  • Integrated Comet browser automation to pre-fill repetitive application fields, reducing manual effort by up to 50%
  • Shared the workflow with friends who adapted it for their own job searches

Tools Used ($0/month)

Apify (free) · n8n (self-hosted) · Gemini API · Google Drive & Sheets · Perplexity Comet (free)

Workflow

Learnings

  • LinkedIn job postings are most effective within 24 hours of posting — many scrapes return stale listings that are already closed
  • Better to scrape directly from company websites rather than LinkedIn for more accurate and complete data
